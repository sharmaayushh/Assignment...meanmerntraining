My Projects

STUDENT ATTENDANCE SYSTEM USING QRCODE


The purpose of this project is to present a system that can scan a student-specific QR code and generate a Microsoft Excel file. The file will show the student's name, E-mail Id and the time of the scan. This model provides an efficient and accurate way to track student attendance. As usual but additionally, the system gives students access to their attendance records, giving them transparency and accountability. This paper presents relevant approaches and field work, details of the proposed system, analysis of the advantages and limitations of QR code technology, and research results and conclusions. The study concludes that the QR code attendance system is a reliable and efficient solution for student attendance tracking.
